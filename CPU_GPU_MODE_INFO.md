# CPU/GPU 模式使用说明

## 功能更新

我们已更新系统以支持在CPU和GPU之间进行选择训练模式：

### 主要功能
1. **设备选择**：在GUI界面中可以选择使用CPU或GPU进行训练
2. **CPU进度条**：为CPU训练添加了详细的进度指示器
3. **实时进度反馈**：CPU训练时显示批次级别的进度更新

### 使用方法
1. 启动GUI界面：`python main.py`
2. 在"计算设备"部分选择"CPU"或"GPU"
3. 配置其他训练参数
4. 点击"开始训练"按钮
5. 训练过程中，CPU模式将显示额外的批次进度条

### 技术细节
- CPU训练时会显示批次级别的进度更新
- GPU训练时使用传统的epoch级别进度更新
- 系统会自动检测GPU可用性并相应调整
- 代码确保了设备选择的正确实现，避免了"用CPU的名字使用GPU"的问题

### 注意事项
- CPU训练通常较慢，但内存限制较小
- GPU训练速度快，但需要足够的显存
- 如果选择GPU但不可用，系统会自动回退到CPU模式
